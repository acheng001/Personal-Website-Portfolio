Previous Projects

As a Computer Science student, I have created multiple projects in the past and on this page, I will include a couple of them!

1) AVL Tree Project

I created this project in one of my CMPT courses.

The purpose of it is it first gets numbers to be put in a tree where each number represents a Node. Afterwards for each new node that is added in or removed, the program manually checks each time to see if the AVL tree is balanced and if it isn't it will self balance!

2) Music Playlist Sorter

This project consists of using linked lists where it has five functions. It can add (A song at a certain position), delete (A song at a certain position, swap the order of the songs, print all of them or quit the program. It was also done in C++ and in summary, the project is a simiplified version of Spotify or iMusic!

3) Image Processor

This project was programmed in Python where it takes an image and can rotate or apply different filters on it. In total there are five different options the user can choose from that modifies the image! An example of one of the modifications is show in the image on the right which transformed the original image to a grayscale filter!
